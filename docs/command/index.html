Command Usage

A comprehensive guide to the available commands in MCEconomy, their syntax, and required permissions.

1. General Commands

These commands are accessible to all players by default.

View Commands 
/economy help
Check Balance 
/economy get <currency>
Transfer Funds 
/economy send <player> <currency> <amount>
Convert to Item 
/economy convert <currency> <amount>

2. Administrative Commands

These commands manage the economy and require specific permissions.

Add Funds mceconomy.add.coin 
/economy add <player> <currency> <amount>
Remove Funds mceconomy.minus.coin 
/economy minus <player> <currency> <amount>
Set Funds mceconomy.set.coin 
/economy set <player> <currency> <amount>
Check Others' Balance mceconomy.get.other 
/economy get <currency> <player>
